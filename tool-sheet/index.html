function names

a function should do exactly what is suggested by its name, no more. have a firm understanding of what a prefix means given the context of the current project. frameworks/libraries may define functions with ultrashort names to improve convenience ie. jQuery defines a function with '$' and the library Lodash has its core function name '_'.
